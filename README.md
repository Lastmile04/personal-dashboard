## How to Use

### Quick Start

1. **Clone the repository:**

git clone https://github.com/Lastmile04/personal-dashboard.git
cd personal-dashboard


2. **Open the project:**

Simply open in your browser
open index.html # macOS
start index.html # Windows


3. **Start adding tasks!**
- Click "Add Task" button
- Enter task details
- Manage your tasks in real-time

### Alternative: Run with Local Server

git clone https://github.com/Lastmile04/personal-dashboard.git
cd personal-dashboard
python -m http.server 8000

Open http://localhost:8000

## Technical Implementation

### Core Features

- **Task Management:** Add, edit, delete operations with real-time UI updates
- **Data Persistence:** localStorage integration for state management across sessions
- **DOM Manipulation:** Efficient DOM traversal and event delegation
- **Responsive Layout:** CSS Grid and Flexbox for adaptive design
- **CSS Animations:** Smooth transitions and visual feedback

### Code Quality

- ~2000+ lines of clean, documented JavaScript
- Modular function structure for maintainability
- Proper error handling and edge case management
- No external dependencies (vanilla JavaScript only)
